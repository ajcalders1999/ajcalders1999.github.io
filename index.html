Proyectos

Aplicación Web – Media Dashboard

Desarrollo de un sistema web en .NET Core 8.0 (ASP.NET MVC) con SQL Server y Entity Framework. La aplicación permite a empresas configurar y visualizar un dashboard interactivo de medios con integración de múltiples APIs open source (clima, tipo de cambio, noticias, etc.), presentando la información mediante widgets personalizables como gráficos, tablas, imágenes y videos. El aplicativo esta realizado en base a streaming de películas y series.

Funcionalidades principales:
* Login y gestión de usuarios con control de roles y permisos.
* Dashboard dinámico y responsive (Bootstrap/Flexbox) compatible con dispositivos móviles.
* CRUD completo para la configuración de widgets (crear, modificar, eliminar, listar).
* Integración de APIs con renderización asíncrona y refresco automático de datos.
* Personalización por usuario: visibilidad de widgets, favoritos en la parte superior, ocultar componentes no deseados.
* Panel de configuración con almacenamiento de API Keys, tiempos de refresco, tamaños y posiciones de widgets.
* Gestión de perfiles de usuario y configuraciones individuales.

Aspectos técnicos destacados:
* Arquitectura modular siguiendo principios SOLID y patrones de diseño.
* Uso de Razor Pages, Middleware, Git y GitHub para control de versiones.
* Código documentado con convenciones de estilo y buenas prácticas.

Aplicación Móvil – Lectura y Escritura de Códigos QR

Desarrollo de una aplicación móvil diseñada para la gestión de activos institucionales mediante la lectura y escritura de códigos QR. La app permite escanear etiquetas asociadas a equipos o recursos, registrar la información y almacenarla de forma estructurada en un archivo Excel. Desde la misma aplicación, el archivo puede ser enviado por correo electrónico, facilitando la distribución de reportes y mejorando la trazabilidad de los activos.

Objetivo principal:
Optimizar el proceso de inventario y seguimiento de activos físicos, reduciendo errores manuales y mejorando la eficiencia en la actualización de registros.

Funcionalidades destacadas:
📷 Escaneo de códigos QR para identificar activos de manera rápida.
📊 Almacenamiento automático de la información en archivos Excel.
📧 Envío de reportes por correo electrónico directamente desde la aplicación.
🔍 Búsqueda y consulta de activos registrados en el sistema.

Tecnologías utilizadas:
* Plataforma de desarrollo móvil: Android Studio.
* Lenguajes: Java.
* Bibliotecas: SDK de escaneo QR, librerías para manejo de Excel, Apache POI.
* Servicios integrados: correo electrónico nativo / SMTP.

Aplicación Web – Gestión del Proceso de Oferta Académica

Desarrollo de una aplicación web en C# con base de datos en MySQL, orientada a la automatización del proceso de matrícula en una Universidad. El sistema facilita la administración de aulas, edificios y horarios, permitiendo asignar automáticamente los cursos a las aulas disponibles de acuerdo con la capacidad y la disponibilidad.
La aplicación incluye la funcionalidad de importar datos de cursos desde archivos Excel, almacenándolos en la base de datos para su posterior gestión. Este proyecto optimiza los procesos internos y contribuye a mejorar la eficiencia operativa de la institución educativa.

Objetivo principal:
Digitalizar y automatizar el flujo de matrícula para reducir el trabajo manual, minimizar errores en la asignación de aulas y garantizar un mejor uso de la infraestructura académica.

Funcionalidades destacadas;
🏫 Gestión de edificios y aulas con su respectiva capacidad.
📅 Administración de horarios para evitar choques entre cursos.
🔄 Asignación automática de cursos a aulas disponibles según capacidad y horario.
📥 Importación de cursos desde Excel y carga directa a la base de datos.
📊 Panel administrativo para consultar, modificar y actualizar datos de matrícula.

Tecnologías utilizadas:
* Lenguaje: C# con WebForms .
* Base de datos: MySQL.
* Integraciones: Librerías para lectura de Excel EPPlus y NPOI.
* Arquitectura: Modelo en capas con separación de lógica de negocio y acceso a datos.
